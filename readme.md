## Setup

**Install Packages**
```
npm i 
```

**Run tests**
```
npm run test
```

**Run locally**
```
npm start
```
**Run docker image**
```
docker build -t aliayman/hackerbay-backend .
docker images
docker run -p 3000:3000 -d aliayman/hackerbay-backend
docker ps
docker logs <container id>
```
